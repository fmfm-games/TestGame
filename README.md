# 遊び方

アップグレードを選んでください．`Enter`で決定．アップグレードは3個選べます．

敵を倒してください．自分は <font color="orange">▲</font> で，敵が <font color="black">▲</font> です．操作方法は以下です．

|入力|操作|
|:--|:--|
|`A`|左旋回|
|`D`|右旋回|
|`Space`|射撃|
|`Left Shift`|ブースト|

## Tips

ブースト中は被弾を無効にします．
